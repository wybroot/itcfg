# 开发日志

## 2026-07-17

### 本次变更主题

完善“组件模板目录 → 组件注册/变量同步 → 环境配置预览”的闭环能力，使系统不再只围绕固定 MVP 组件运行，而是可以从模板目录动态发现、同步和查看组件模板。

### 后端变更

- 模板管理 API 增强
  - 新增 `GET /api/v1/templates/:name/variables`，用于读取指定模板的变量定义。
  - 新增 `POST /api/v1/templates/sync`，用于将模板目录同步为组件，并支持按模板名选择同步、是否覆盖变量。
  - `GET /api/v1/templates` 返回信息扩展：模板目录、变量分组、配置文件、依赖、注册状态、关联组件 ID、启用状态。

- 组件同步逻辑增强
  - 新增 `ComponentService.SyncTemplate`，可根据模板 `manifest.yaml` 创建或更新组件，并同步 `variables.yaml` 中的变量。
  - 抽取 `buildVariablesFromTemplate`，复用模板变量解析、类型规范化和校验逻辑。
  - 原“从模板导入变量”继续使用同一套变量构建逻辑，避免同步和导入行为不一致。

- 初始化种子组件逻辑调整
  - `SeedComponents` 从固定的 `postgresql/java-app/nginx` 改为扫描模板目录下所有包含 `manifest.yaml` 的组件模板。
  - 支持通过环境变量 `SEED_TEMPLATES` 控制初始化范围：
    - 空值或 `all`：同步全部模板。
    - `none`：跳过模板同步。
    - 逗号分隔列表：只同步指定模板目录。

- 配置预览增强
  - `PreviewConfigs` 支持接收前端当前未保存的 `values`，预览时会覆盖数据库中已保存配置，便于“边填边预览”。

- 模板元信息增强
  - `TemplateInfo` 增加模板目录、变量分组、配置文件清单、依赖、注册状态等字段。
  - Redis 模板变量说明中的引号格式调整为合法 YAML 字符串。

### 前端变更

- API 封装
  - 新增 `getTemplateVariables(templateName)`。
  - 新增 `syncTemplates(data)`。
  - `previewConfigs` 增加 `values` 参数，用于传入当前表单值进行实时预览。

- 组件管理页
  - 操作入口调整为“从模板添加组件”和“新建自定义组件”。
  - 组件列表增加描述展示和变量分组展示。
  - 变量抽屉改为按分组展示变量，突出“同步模板变量 / 覆盖同步”，手工新增变量被标记为高级操作。
  - 变量表格增加说明、关联变量等信息。

- 模板目录页
  - 页面重构为统一的 `PageHeader` + `PageCard` 布局。
  - 展示模板名称、目录、分类、变量分组、依赖、变量数、配置文件数、注册状态。
  - 支持刷新、同步全部模板、单个模板导入/同步。
  - 新增模板详情抽屉，可查看变量定义和配置文件输出清单。

- 环境组件选择
  - 不再限制为固定的 `postgresql/java-app/nginx`，改为展示所有启用组件。
  - 增加变量数量展示，并按分类和显示名排序。
  - 空状态文案提示先到模板目录同步组件。

- 环境配置页
  - 组件摘要增加描述展示。
  - 增加必填项完成度统计。
  - 抽取 `cleanCurrentValues`，保存和预览共用同一套过滤逻辑。
  - 配置预览会带上当前表单值，无需先保存即可看到渲染结果。

### 影响范围

- 后端：`handler`、`service`、`repository/seed`、`template engine`。
- 前端：模板目录、组件管理、环境组件选择、环境配置预览。
- 模板：Redis 变量 YAML 描述字段。
